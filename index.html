Archivio e compatibilità

Formato supportato

XLSX BNL con intestazione simile a quella del file che hai caricato: tabella a partire dalla riga 9, colonne Data contabile, Data valuta, Descrizione, Dettaglio, Importo.

Persistenza locale

Categorie, regole e ultimo dataset vengono salvati nel browser. Su iPad/Safari è meglio usare sempre la stessa URL GitHub Pages.

Esportazione

Il file esportato aggiunge colonne Merchant, Categoria, Sottocategoria, Tipo movimento, Note.
