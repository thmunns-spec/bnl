# BNL FinImport v2

## Cosa fa
- carica automaticamente i movimenti inclusi in `data.json`
- permette anche import manuale di file `.xlsx` o `.csv`
- categorizza ogni movimento manualmente
- salva categorie e assegnazioni nel browser
- esporta un CSV finale con la colonna `Categoria`

## Pubblicazione su GitHub Pages
Carica **tutti** questi file nella root della repository:
- `index.html`
- `styles.css`
- `app.js`
- `data.json`

Poi attiva GitHub Pages sulla branch principale.

## Nota pratica
Se premi il bottone di import e non succede niente, guarda il messaggio in alto:
questa versione mostra sempre uno stato chiaro invece di restare muta.
